# code-check

Инструменты для проверки кода выполненных заданий.

### Как подключить

```
    repositories {
        maven {
            url "https://github.com/csu-java-courses/repo/raw/master/"
        }
    }

    dependencies {
        compile group: 'org.example', name: 'code-check', version: '1.1'
    }

```
